# LGS_Turkce_Soru_Tahminleme_Modeli
Bu projede, geçmiş yıllara ait LGS Türkçe soruları kullanılarak yeni soruların türünü tahmin eden bir makine öğrenmesi modeli geliştirilmiştir. TF-IDF tabanlı özellik çıkarımı ve Naive Bayes sınıflandırıcısı kullanılmıştır. Veri temizleme, ön işleme, model eğitimi, görselleştirme ve performans değerlendirme adımlarını içeren bir NLP projesidir.
